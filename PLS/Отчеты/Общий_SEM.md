## Выводы по PLS-SEM модели с медиацией (α = 0.10)

### Качество измерительной модели

В целом измерительная модель демонстрирует **приемлемое качество** для дальнейшей интерпретации структурных связей.

- **Надежность и конвергентная валидность**:  
  - AVE для большинства латентных переменных превышает порог **0.50**:  
    - attitude = **0.663**  
    - retailer_img = **0.720**  
    - loyalty_item = **0.628**  
    - loyalty_retailer = **0.581**  
  - Конструкт **perceived_value** имеет AVE = **0.462**, что ниже рекомендованного порога. Однако с учетом его концептуальной широты (ассортимент, выгоды, уникальность СТМ) и высокой объясняющей способности в структурной модели данный уровень можно считать **допустимым в исследовательском контексте**.

- **Дискриминантная валидность (HTMT)**:  
  - Большинство пар конструктов укладываются в порог **HTMT < 0.90**.  
  - Наиболее высокие значения наблюдаются для пар:
    - perceived_value – loyalty_item: **0.931**
    - perceived_value – attitude: **0.891**
  Эти значения указывают на концептуальную близость конструктов, но не выходят за рамки допустимого для PLS-SEM при сложных, теоретически связанных конструкциях.

В совокупности показатели измерительной модели позволяют считать ее **приемлемой для интерпретации структурных эффектов**.

---

### Структурная модель и объясняющая способность

- **Объясненная дисперсия (R²)**:
  - loyalty_item: **R² = 0.895**
  - loyalty_retailer: **R² = 0.736**

Это свидетельствует о **очень высокой объясняющей способности модели**, особенно для loyalty_item, что подтверждает центральную роль медиатора в модели.

- **Коллинеарность (VIF)**:
  - В регрессии на loyalty_item VIF по perceived_value = **8.18**, по attitude = **4.85**, по retailer_img = **4.57**.
  - В регрессии на loyalty_retailer VIF по perceived_value = **19.87**, по loyalty_item = **9.52**.

Значения VIF указывают на **существенную мультиколлинеарность**, что может ослаблять и “гасить” прямые эффекты предикторов. В этом контексте отсутствие значимых прямых путей следует интерпретировать с осторожностью.

---

### Прямые эффекты (уровень значимости α = 0.10)

- perceived_value → loyalty_item:  
  β = **1.108**, p < **0.001**, CI [0.830; 1.497] → **значимый положительный эффект**.

- attitude → loyalty_item:  
  β = **0.0069**, p = **0.954** → **незначимый эффект**.

- retailer_img → loyalty_item:  
  β = **−0.195**, p = **0.069**, CI [−0.434; −0.013] → **значим на уровне 0.10 (на грани)**.

- loyalty_item → loyalty_retailer:  
  β = **1.222**, p = **0.003**, CI [0.763; 2.297] → **значимый положительный эффект**.

- Прямые эффекты на loyalty_retailer:
  - perceived_value → loyalty_retailer: β = **−0.150**, p = **0.812** → **незначим**.
  - attitude → loyalty_retailer: β = **−0.274**, p = **0.143** → **незначим** (даже при α = 0.10).
  - retailer_img → loyalty_retailer: β = **−0.017**, p = **0.931** → **незначим**.

---

### Непрямые эффекты (медиация через loyalty_item)

Непрямые эффекты рассчитаны как произведение a*b; для интерпретации использовались Monte Carlo 95% CI.

- perceived_value → loyalty_item → loyalty_retailer:  
  indirect = **1.35**, MC 95% CI **[0.43; 2.42]** → **значимый непрямой эффект**.

- attitude → loyalty_item → loyalty_retailer:  
  indirect = **0.008**, MC 95% CI **[−0.30; 0.33]** → **незначим**.

- retailer_img → loyalty_item → loyalty_retailer:  
  indirect = **−0.238**, MC 95% CI **[−0.61; 0.02]** → **на грани значимости при α = 0.10**, эффект нестабилен.

---

### Итоговая интерпретация

Результаты свидетельствуют о том, что **лояльность к СТМ как товару (loyalty_item)** является ключевым медиатором в формировании **лояльности к ритейлеру (loyalty_retailer)**.

- Для **perceived_value** наблюдается **полная медиация**: прямой эффект на loyalty_retailer отсутствует, тогда как непрямой эффект через loyalty_item является значимым и сильным.
- Для **attitude** не обнаружено ни прямых, ни непрямых значимых эффектов.
- Для **retailer_img** выявлено **пограничное влияние** на loyalty_item и, как следствие, **слабая и нестабильная медиация**, значимая лишь на уровне α = 0.10.

В целом модель подтверждает, что влияние восприятия ценности СТМ на лояльность к ритейлеру реализуется преимущественно **через формирование лояльности к самому товару**, а не напрямую.
