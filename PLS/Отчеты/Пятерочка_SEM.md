## Выводы по PLS-SEM модели для ритейлера «Пятерочка» (α = 0.10)  
*Сравнение с общей (агрегированной) моделью*

В данном разделе представлены результаты PLS-SEM анализа, проведенного **на данных ритейлера «Пятерочка»**, а также их сопоставление с выводами **общей модели по всем ритейлерам**. Уровень значимости принят **α = 0.10**; эффекты, находящиеся вблизи порога, обозначаются как *«на грани значимости»*.

---

## 1. Качество измерительной модели

### Конвергентная валидность (AVE)

- perceived_value = **0.464**  
- attitude = **0.674**  
- retailer_img = **0.715**  
- loyalty_item = **0.647**  
- loyalty_retailer = **0.598**

По сравнению с общей моделью показатели AVE для «Пятерочки» находятся **на сопоставимом уровне**. Конструкт **perceived_value** снова демонстрирует значение ниже 0.50, однако отклонение умеренное и систематически воспроизводится во всех спецификациях. С учетом агрегированной природы данного конструкта и его центральной роли в модели, конвергентную валидность можно считать **условно приемлемой**.

### Дискриминантная валидность (HTMT)

- HTMT(perceived_value – loyalty_item) = **0.949**
- HTMT(perceived_value – retailer_img) = **0.905**
- HTMT(perceived_value – attitude) = **0.836**

В модели для «Пятерочки» наблюдается **наиболее высокая концептуальная близость** perceived_value и loyalty_item по сравнению с другими ритейлерами. Значения HTMT вплотную приближаются к порогу 0.95, что указывает на **сильное пересечение содержания конструктов**, однако не выходит за критический диапазон. В исследовательском PLS-контексте дискриминантная валидность оценивается как **погранично допустимая**.

---

## 2. Объясняющая способность модели

- **R² loyalty_item = 0.943**
- **R² loyalty_retailer = 0.634**

По сравнению с общей моделью:
- объясняющая способность для loyalty_item **еще выше**, что указывает на почти полное объяснение вариации лояльности к СТМ,
- R² для loyalty_retailer **заметно ниже**, чем в агрегированной модели, но остается на **достаточно высоком уровне**.

## 3. Коллинеарность (VIF)

- В регрессии на loyalty_item:
  - perceived_value: **VIF = 7.31**
- В регрессии на loyalty_retailer:
  - perceived_value: **VIF = 31.01**
  - loyalty_item: **VIF = 17.50**

По сравнению с общей моделью коллинеарность в «Пятерочке» **наиболее выражена**, особенно в уравнении для loyalty_retailer. Это указывает на сильное перекрытие объясняющей информации между perceived_value и loyalty_item и создает условия для **статистического подавления прямых эффектов**.

## 4. Структурные пути (α = 0.10)

### Пути к loyalty_item (a-пути)

- perceived_value → loyalty_item:  
  β = **1.164**, p < **0.001** → **значимый положительный эффект**.
- attitude → loyalty_item:  
  β = **0.134**, p = **0.325** → **незначим**.
- retailer_img → loyalty_item:  
  β = **−0.355**, p = **0.047** → **значим при α = 0.10 (на грани)**.

В сравнении с общей моделью и другими ритейлерами, в «Пятерочке» проявляется **пограничный отрицательный эффект имиджа ритейлера на лояльность к СТМ**, что может отражать специфику восприятия СТМ в дискаунтерском формате.

### Пути к loyalty_retailer (b- и c′-пути)

- loyalty_item → loyalty_retailer:  
  β = **1.229**, p = **0.308** → **незначим**.
- perceived_value → loyalty_retailer:  
  β = **−0.661**, p = **0.685** → **незначим**.
- attitude → loyalty_retailer:  
  β = **−0.246**, p = **0.483** → **незначим**.
- retailer_img → loyalty_retailer:  
  β = **0.479**, p = **0.479** → **незначим**.

В отличие от общей модели, где путь loyalty_item → loyalty_retailer был значим, в «Пятерочке» данный эффект **не подтверждается**.

---

## 6. Сравнительный итог

Сопоставление с общей моделью показывает, что:

- В агрегированной модели наблюдалась **устойчивая полная медиация** через loyalty_item.
- В модели для «Пятерочки»:
  - perceived_value стабильно формирует loyalty_item,
  - однако дальнейшая трансляция в loyalty_retailer **ослабляется**,
  - высокая коллинеарность приводит к **размыванию как прямых, так и непрямых эффектов**.

### Итоговый вывод для «Пятерочки»

Для ритейлера **«Пятерочка»** воспринимаемая ценность СТМ является **ключевым фактором формирования лояльности к СТМ как товару**, однако данный эффект **не трансформируется устойчиво в лояльность к ритейлеру**. В отличие от общей модели, механизм медиации через loyalty_item **не подтверждается эмпирически**, что указывает на возможную специфику поведения покупателей «Пятерочки», для которых лояльность к СТМ не обязательно означает лояльность к самой сети.

Полученные результаты подчеркивают, что **агрегированные эффекты не полностью воспроизводятся на уровне дискаунтерского формата**, что подтверждает необходимость раздельного анализа по ритейлерам.
