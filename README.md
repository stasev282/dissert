# Академическое исследование: CFA → EFA → PLS‑SEM (медиация)

## Цель исследования
Этот репозиторий содержит воспроизводимый анализ для академического исследования уровня магистратуры. Цель — проверить исследовательскую модель, включающую измерительную часть (CFA/EFA) и структурные связи в PLS‑SEM с возможной медиацией.

## Исследовательская модель
Анализ строится в следующей логике:
1. **CFA (Confirmatory Factor Analysis)** — проверка первоначальной измерительной модели.
2. **EFA (Exploratory Factor Analysis)** — уточнение факторной структуры при необходимости.
3. **PLS‑SEM** — оценка структурной модели, включая проверку прямых и косвенных (медиирующих) эффектов.

## Данные и этика
Используются исходные данные респондентских измерений в табличном формате. **Сырые данные не публикуются** в открытом репозитории по причинам NDA и этических ограничений. В репозитории присутствуют только код, описание этапов анализа и агрегированные результаты.

## Структура репозитория
- `Факторный анализ/` — скрипты и материалы для CFA/EFA.
- `PLS/` — скрипты PLS‑SEM.
- `По кластерам/` — анализ по кластерам (подвыборки).
- `По полу/` — анализ по полу (подвыборки).
- `*.R` — основные скрипты анализа.
- `*.Rmd` — отчётные документы (при наличии).

## Воспроизведение анализа (без доступа к данным)
1. Установить R и необходимые пакеты (например, `lavaan`, `psych`, `cSEM`, `tidyverse`).
2. Поместить исходные данные в локальную папку данных (см. `.gitignore`), не публикуя их в репозиторий.
3. Последовательно запустить скрипты CFA → EFA → PLS‑SEM в соответствующих папках.
4. Сформировать отчёты/таблицы результатов с использованием скриптов проекта.

## Результаты на выходе
В рамках анализа формируются:
- показатели качества измерительных моделей (валидность, надежность, соответствие модели);
- итоговая факторная структура (EFA) и её интерпретация;
- параметры структурной модели PLS‑SEM (коэффициенты путей, медиационные эффекты);
- таблицы, графики и промежуточные отчёты по этапам.

## Примечание о доступе к данным
Доступ к исходным данным предоставляется только в рамках согласованных условий (NDA/этические ограничения). В публичной версии репозитория данных нет.
