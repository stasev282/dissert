# Вывод по результатам анализа гендерной модерации косвенных эффектов  
(таблица `pls_2stage_gender_indirect_diffs`)

В таблице представлены результаты проверки гендерной модерации косвенных эффектов во второй стадии PLS-SEM. Анализ основан на разностях косвенных эффектов между группами по полу, рассчитанных как difference of indirect effects, с оценкой статистической значимости на основе p-value и бутстрэп-доверительных интервалов.

## Доказательства отсутствия гендерной модерации

Для косвенного эффекта `perceived_value → loyalty_retailer` разность между мужской и женской группами составляет -6.91 (столбец `estimate`), при 95% бутстрэп-доверительном интервале [-36.88; 24.93] (столбцы `ci_low`, `ci_high`) и p-value = 0.5981 (столбец `p_value`). Доверительный интервал включает 0, а значение p-value существенно превышает порог 0.05, что указывает на отсутствие статистически значимого различия косвенных эффектов между гендерными группами.

Для косвенного эффекта `attitude → loyalty_retailer` разность между гендерными группами составляет -0.62 при 95% CI [-7.57; 6.19] и p-value = 0.8684. Интервал перекрывает 0, p-value значительно выше 0.05, что свидетельствует об отсутствии гендерной модерации данного косвенного эффекта.

Для косвенного эффекта `retailer_img → loyalty_retailer` разность между мужской и женской группами составляет -2.12 при 95% CI [-15.42; 11.18] и p-value = 0.7486. Статистически значимого различия между группами не зафиксировано, так как доверительный интервал включает 0, а p-value превышает 0.05.

## Итоговый вывод

По всем проверенным косвенным эффектам (`perceived_value`, `attitude`, `retailer_img`) различия между гендерными группами статистически незначимы. Во всех случаях 95% бутстрэп-доверительные интервалы включают 0, а значения p-value превышают порог 0.05. Это подтверждает отсутствие гендерной модерации косвенных эффектов и указывает на инвариантность медиационных механизмов модели по полу.
